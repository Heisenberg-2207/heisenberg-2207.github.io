# Results

Figures generated by the `Evaluator` class (see [`pipeline/src/QEM/`](../pipeline/src/QEM/readme.md)) and related analysis notebooks, comparing ML-based mitigation against the Noisy and ZNE (Linear/Richardson) baselines.

| File | Description |
|---|---|
| `sample_circuit.png` | Example transpiled quantum circuit used for data generation, showing native gates (`RZ`, `√X`, `Ecr`) and measurement layout. |
| `ann.png`, `cnn.png`, `rnn.png` | MAE-comparison bar chart + predicted-vs-ideal scatter plot (Noisy / Linear ZNE / Richardson ZNE / Predicted) for the ANN, CNN, and RNN models respectively. |
| `rf.png`, `rf1.png`, `rf2.png` | Random Forest model results — overall and per-run/per-configuration breakdowns. |
| `xgb.png`, `xgb1a.png`, `xgb1b.png`, `xgb1c.png` | XGBoost model results — overall and per-run/per-configuration breakdowns. |
| `kan1a.png`, `kan1b.png` | KAN (Kolmogorov-Arnold Network) model results — per-qubit RMSE breakdown and noisy-vs-mitigated distance-to-ideal distributions. |
| `q10_d2.png`, `q10_d10.png` | Results on 10-qubit circuits at shallow (depth 2) vs. deeper (depth 10) circuit depth — used to study how mitigation accuracy scales with circuit depth. |
| `net1.png` | Network architecture / training diagram for one of the deep models. |
| `hardware_models.png` | GCN vs. GAT vs. GKAN comparison (MAE, "Ideal vs Predicted" vs. "Ideal vs Noisy") on real **IBM Q Brisbane** hardware data. |

See the [hardware/](../hardware/README.md) and [pipeline/src/QEM/](../pipeline/src/QEM/readme.md) folders for the code that produced these figures.
